Metric / Imperial Converter

Converter Request


OR 
GET /api/converter?input={{input}}

Example

GET /api/converter?input=123mi

Supported units: km, mi, L, gal, kg, lbs

Converter Response

200 Ok
{
    initNum: 123,
    initUnit: "mi",
    returnNum: 197.94882,
    returnUnit: "km",
    string: "123 miles converts to 197.94882 kilometers"
}
